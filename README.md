# wiz-tools

A collection of small projects, scripts, and tools built while working with [Wiz](https://www.wiz.io). Shared here in case they're useful to others.

## Tools

### [wiz-shi-report-viewer](wiz-shi-report-viewer/)

A zero-dependency, single-file web application that transforms Wiz security issue export CSVs into an interactive report viewer.

**Key features:** interactive table (flat + grouped views), severity/source filtering, full-text search, presentation mode, per-issue notes with localStorage persistence, Wiz Graph Query links, dark/light theme.

**Usage:** open `wiz-shi-report-viewer/wiz-shi-report-viewer.html` in any modern browser and upload a Wiz CSV export. No install or build step required.

See the [tool README](wiz-shi-report-viewer/README.md) for full documentation.
